# 🌍 Common Network Devices

## Router

The router manages internet connectivity and directs traffic between devices.

## Modem

The modem connects the local network to the internet service provider (ISP).

## Switch

A switch connects multiple devices within the same network.

## Wireless Access Point

Provides wireless connectivity to devices.

---

# 📶 Wi-Fi Setup Basics

Recommended steps:

1. Connect the router correctly.
2. Access the router management page.
3. Change the default administrator password.
4. Configure Wi-Fi name (SSID).
5. Enable WPA2 or WPA3 security.
6. Test connectivity.

# 🌐 Basic TCP/IP Concepts

Important network information:

- IP Address
- Subnet Mask
- Default Gateway
- DNS Server

Example:

IP Address: 192.168.1.100

Subnet Mask: 255.255.255.0

Default Gateway: 192.168.1.1
